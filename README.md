## SSH 连接教程：怎么登录你的 VPS

服务器跑起来之后，你需要通过 SSH 远程连接进去管理。

### Windows 用户

Windows 10 和 11 自带 SSH 客户端，不需要额外安装软件。

1. 打开"命令提示符"（Win + R → 输入 cmd）
2. 输入命令：

bash
ssh root@你的服务器IP


3. 第一次连接会问你是否信任这台服务器，输入 `yes` 确认
4. 然后输入邮件里收到的密码，回车

> 提示：输密码时光标不动是正常现象，Linux 的密码输入默认不显示字符，打完回车就行。

如果你更喜欢图形界面，可以装个 **MobaXterm** 或 **PuTTY**，界面更友好。

### Mac / Linux 用户

直接打开终端，命令和上面一样：

bash
ssh root@你的服务器IP


### 连接成功之后

登进去你会看到命令行提示符，大概长这样：


root@vm-xxxxx:~#


到这里，服务器就在你手里了，可以开始做任何你想做的事——建网站、跑 Bot、搭数据库、部署应用，随便。
